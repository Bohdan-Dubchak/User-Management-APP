# User-Management-APP

Сучасний веб-додаток для управління користувачами з можливістю створення, редагування та видалення записів.

Особливості:
- ✨ Чистий JavaScript ES6+ без фреймворків
- 🏗️ Модульна архітектура
- 🚀 Швидка робота та оптимізація
- 📱 Адаптивний інтерфейс

**Створення користувача:**
- Заповнення форми з валідацією
- Вибір аватара зі списку
- Миттєве відображення нового користувача

**Редагування користувача:**
- Відкриття модального вікна
- Зміна даних користувача
- Збереження змін

**Видалення користувача:**
- Підтвердження дії
- Видалення з анімацією


### Валідація:
- **Ім'я:** 2-23 символи
- **Місто:** 2-20 символів
- **Email:** валідний формат
- **Аватар:** обов'язкове поле

### Frontend:
- **JavaScript ES6+** - основна мова програмування
- **HTML5** - розмітка
- **CSS3** - стилізація з CSS Variables
- **Fetch API** - HTTP запити

## Backend/API:
- **MockAPI.io** - тестовий REST API

### Архітектурні паттерни:
- **Module Pattern** - організація коду
- **Observer Pattern** - управління станом (Store)
- **Event Delegation** - оптимізація подій
